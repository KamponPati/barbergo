# Dispute Guide (Ops/Admin)

Last updated: 2026-02-16

## Intake

- Verify booking id and actor identity
- Check booking state transition eligibility
- Collect evidence note and timeline references

## Resolution Steps

- classify dispute type
- evaluate policy eligibility
- apply action (refund/deny/manual follow-up)
- record final decision in audit trail

## Escalation

- high-risk cases follow incident escalation matrix
- unresolved disputes beyond SLA escalate to operations lead
