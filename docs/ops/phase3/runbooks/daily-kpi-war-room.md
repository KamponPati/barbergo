# Phase 3 Daily KPI War Room

## Cadence
- Frequency: daily, 09:30 local time
- Duration: 30 minutes
- Participants: Ops lead, Product, Eng on-call, Support lead

## Daily Inputs
- Reconciliation report (gateway vs internal ledger)
- SLA dashboard: confirm time, on-time start, completion rate
- Quality dashboard: complaint, refund, repeat rate
- Open incidents and unresolved disputes

## Review Agenda
1. Yesterday KPI snapshot vs thresholds
2. Zone-level anomalies and partner outliers
3. Incidents/postmortem carry-over actions
4. Decision log and owner assignment

## KPI Thresholds
- Avg confirm time <= 10 min
- On-time start rate >= 90%
- Completion rate >= 92%
- Complaint rate <= 3%
- Refund rate <= 2%
- Repeat rate >= 25%
