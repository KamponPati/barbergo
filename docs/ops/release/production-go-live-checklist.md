# Production Go-Live Checklist

- [ ] CI green: lint/test/build
- [ ] Staging e2e smoke passed
- [ ] Security workflow passed (SAST, deps, secrets)
- [ ] Perf smoke passed against staging target

## Data and Reliability
- [ ] Migration safety checks passed
- [ ] Backup and restore drill evidence updated
- [ ] Rollback command validated

## Operations
- [ ] On-call schedule published for launch week
- [ ] Escalation matrix reviewed
- [ ] Communication templates prepared

## Product and Support
- [ ] Launch comms approved
- [ ] Support playbook updated
- [ ] Success metrics dashboard ready

- [ ] Product Lead
