# BarberGo Brand Kit (Phase 9)

Last updated: 2026-02-16

## Brand Core

- Brand name: `BarberGo`
- Positioning: booking haircut as easy as ride-hailing
- Promise: fast, clear, reliable
- Primary audiences:
  - customers
  - partner shops
  - operations/admin

## Visual Direction

- Primary palette:
  - `--brand`: `#0f766e`
  - `--brand-strong`: `#0b4f56`
  - `--bg-main`: `#f4f2ea`
  - `--ink`: `#1f2937`
- Typography:
  - display/body: `Space Grotesk`
  - code/data: `IBM Plex Mono`
- Shape system:
  - soft rounded corners (`--radius-xs` to `--radius-lg`)
  - subtle elevation shadows (`--shadow-sm`, `--shadow-md`)

## Logo and Illustration Guidelines

- Current phase uses text-first logo lockup (`BarberGo` wordmark) for speed.
- Icon direction for production:
  - simple monoline barber mark
  - consistent stroke width
  - optimized for app icon and favicon
- Illustration style:
  - flat editorial shapes
  - warm neutral backgrounds with teal accents
  - avoid photorealistic mixed styles

## Tone of Voice

- Thai: กระชับ ตรงประเด็น สุภาพ และใช้งานจริง
- English: concise, operational, trustworthy
- Copy rules:
  - avoid jargon in customer-facing screens
  - keep status/error messages actionable
  - use role-specific wording (customer/partner/admin)

## Content Consistency Rules

- Use one canonical term per concept:
  - `booking`
  - `queue`
  - `dispute`
  - `wallet`
- Keep TH/EN parity for core labels and key user actions.
